#Webbell is a internet controlled helicopter

Web = the internet

Bell = a helicopter manufacturer



###to run locally

    $ brew install node
    $ curl http://npmjs.org/install.sh | sh
    $ npm install
    $ gem install foreman
    $ foreman start
